🔒 Security Architecture — How VIBE Is Locked Down

🌐
Layer 1: Edge
Cloudflare WAF
DDoS protection, rate limiting, bot filtering before any request reaches your server. Same Cloudflare you use for EXIAL.
🔐
Layer 2: Auth
JWT + bcrypt
Same JWT pattern as NaluAsk. Every API request requires a valid token. Signed with a server-side secret, expires on schedule.
🎫
Layer 3: Entitlements
Entitlement Table
Every playback request checks the entitlements table. Subscription status alone does NOT grant access. Entitlement must be active.
🔑
Layer 4: Playback
Signed Tokens (TTL 4h)
Backend generates a short-lived signed URL from the provider API (Mux/Vimeo). Raw video URLs are never exposed. Tokens expire and can't be reused.
🏦
Layer 5: Payments
Stripe Webhooks + HTTPS
Stripe webhook signature verified server-side before any entitlement is created. HTTPS enforced via Cloudflare SSL. No card data ever touches your server.
